Git Made Human

Stop memorizing Git commands. Use plain English instead. 

$ omg create branch feature
Branch 'feature' created successfully

$ omg switch to branch feature
Switched branch 'feature'

$ omg list branches
* feature
  main
  develop

Installation

One-Line Install (Recommended) 

curl -sSL https://raw.githubusercontent.com/imcodekhan/oh-my-git/master/install.sh | bash

The installer will:

  • Download Oh-My-Git scripts
  • Set up shell aliases
  • Configure your environment

Manual Setup

If you prefer to set up Oh-My-Git manually, follow these steps: 

# Clone repository
git clone https://github.com/imcodekhan/oh-my-git.git ~/.oh-my-git

# Add to shell config (choose one)
echo 'source ~/.oh-my-git/oh-my-git.sh' >> ~/.bashrc
echo 'source ~/.oh-my-git/oh-my-git.sh' >> ~/.zshrc

# Add alias
echo 'alias omg="oh-my-git"' >> ~/.bashrc  # or .zshrc

# Reload shell
source ~/.bashrc  # or source ~/.zshrc

Commands

Branch Management

omg create branch <name> Create a new branch
omg switch branch <name> Switch to a branch
omg list branches Show all branches
omg delete branch <name> Delete a branch
